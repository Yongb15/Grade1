배경 : 전경색 #e4d99c 배경색 #fafb7c 사용 방사형 그레디언트도구로 선택후 가운데서 드래그

1. 배경 꾸미기
배경이 심심해보여 사용자 정의 모양도구 를 사용하여 배경에 삽입
백터 마스크사용 후 칠

2. 우리 학교를 소개합니다.
수평문자도구로 '우리 학교를  소개합니다!'를 입력 -> 이 글자를 강조하기 위해 레이어 스타일 획(25px, 바깥쪽, 흰색으로 변경)

3. 학교 사진
1) 사진 액자 : 새 레이어 추가 -> 사각형 선택 운곽 도구 사용 -> 페인트통 도구를 전경색에서 패턴(풀사파리)로 변경 후 칠-> 그림자로 입체감을 표현하기 위해 레이어 스타일(경사와 엠보스)사용
2) 학교사진열기 후 레이어 추가 클러핑마스크로 표현할 곳 새 레이어 추가 후 검은색으로 칠하고 학교사진 클러핑 마스크 적용
3) 수평문자도구로 '유한대학교'입력 -> 강조를 위해 레이어 스타일 획(크기 20px)적용
4) 수평문자도구로 유한대학교소개를 입력 -> 살짝 강조를 위해 레이어 스타일 획(크기 14px)적용

5) 유한대학교 약도
1) 사진 액자 : 새 레이어 추가 -> 사각형 선택 운곽 도구 사용 -> 페인트통 도구를 전경색에서 패턴(풀사파리)로 변경 후 칠-> 그림자로 입체감을 표현하기 위해 레이어 스타일(경사와 엠보스)사용
2) 수평문자도구로 '유한대학교 약도'입력 -> 강조를 위해 레이어 스타일 획(크기 20px)적용
3) 수평문자도구로 유한대학교 약도 내용을 입력 -> 살짝 강조를 위해 레이어 스타일 획(크기 10px)적용

6) 유일한 박사님 스케치로 만들기
1) 사진 열기 -> 유일한박사님1 레이어의 채도 감소 -> 유일한 박사님 레이어 복사 -> 원본 레이어에 필터 적용 -> 브러쉬 획(그물눈(6, 5, 1))로 적용 후 칠하기 -> 지우개로 불필요한 부분 지우기
-> 복사한 레이어에 필터 적용( 가장자리 광선 효과 (1, 2, 5)적용 -> 이미지 반전 -> 복사한 레이어에 레이어 마스크 추가 -> 검은색 브러쉬로 칠
3) 수평문자도구로 '유일한 박사'입력 -> 강조를 위해 레이어 스타일 획(크기 20px)적용
4) 수평문자도구로 유한한박사 소개 내용을 입력 -> 살짝 강조를 위해 레이어 스타일 획(크기 14px)적용
