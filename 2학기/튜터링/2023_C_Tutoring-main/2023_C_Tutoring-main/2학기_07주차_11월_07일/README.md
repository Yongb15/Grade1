# '연결리스트 만들기' 보충 설명

* ##### while(getchar() != '\n'); 

while(getchar() != '\n'); 라는 것으로 입력버퍼를 비워주고 있는데, 이 명령의 뜻은 다음과 같다.  
1. getchar()함수로 버퍼에 남아있는지 없는지를 확인한다.
2. \n과 비교를 한다. 
3. 비교를 하여서 \n이 아니면 남아있는 것을 계속 읽어들이면서 비워낸다. 
4. 조건이 맞지 않기 때문에 반복문을 종료한다.

이 말을 코드로 다시 풀어 쓰면 다음과 같다.

```swift
char a;

while (1)
{
    a = getchar();
	
    if (a == '\n')
        break;
}
```
